## simDHT
A DHT crawler, written in Python, the code is very very simple.

## Start simDHT
```bash
python simDHT.py
```

## Dependencies
1. [bencode](https://pypi.python.org/pypi/bencode/1.0)

## Notice
You can edit `max_node_qsize` value for speed.
